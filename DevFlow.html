Set up

  • run npm i react-router-dom in the front end
  • Run 'rafce' in App.js after everything is cleared
  • To return JSX in App.js, replace the { } with ( )
  • Stop the server. Install react-router-dom in the frontend
  • Delete everything in index.css
  • Delete App.css and App.test
  • App.js imports all of the routes
  • If you are returning more than one element in a component, put them in { with a return() statment.}
  • In index.js, import BrowserRouter and wrap just the App component in it. Otherwise you'll throw an error.
